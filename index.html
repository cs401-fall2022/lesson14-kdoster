Typescript

Introduction

Typescript is a language similar to Javascript. It takes the features that Javascript has to offer, and makes it more "strict," but not in the sense of "strict functions." Like the name implies, it adds the strictness into the typing of variables. Javascript is very loose on asserting that you use consistent typing, but Typescript enforces it, making code writing, reading, and debugging much clearer and easier. It also makes it much closer to Object Oriented Programming Languages like Java and C++.
